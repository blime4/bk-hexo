[官网下载](http://mimosa-pudica.net/zsh-incremental.html)

```
  cd .oh-my-zsh/plugins/
  mkdir incr
  //将下载好的文件放在这个文件夹中
  source incr*.zsh
```


然后在./zshrc中添加  source ~/.oh-my-zsh/plugins/incr/incr*.zsh

```
  sudo vim ~/.zshrc
  source ~/.zshrc
```
