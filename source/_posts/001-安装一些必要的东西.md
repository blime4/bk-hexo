## 安装pip

```
sudo apt-get install python3-pip
pip3 --version
```

## 安装matplotlib

```
pip3 install -U matplotlib
```

## 安装seaborn
```
pip3 install seaborn
```

## 安装brewer2mpl
```
pip3 install brewer2mpl
```

## 安装jupyter notebook
```
pip3 install jupyter
```
